Carga masiva de Colaboradores

Crea y actualiza usuarios desde un archivo Excel

1

Descarga la plantilla

Columnas obligatorias: nombre, email

Columnas opcionales: dni, codigoEmpleado, area, cargo, telefono, direccion, rol, emailCoordinador, banco, numeroCuenta, cci, tipoCuenta

El rol admite: Colaborador, Coordinador, Contabilidad o Administrador (por defecto Colaborador). La contraseña se genera automáticamente; podrás descargarla al finalizar.

¿Ya tienes usuarios? Descarga la plantilla con los usuarios actuales, edítala y vuelve a subirla: las filas con id se actualizan y las filas sin id se crean. No borres la columna id. Una celda vacía no borra el dato de un usuario existente.
